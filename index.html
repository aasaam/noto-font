aasaam Noto

آسام نوتو

aasaam Noto is same Google Noto font but modified and optimized for using in web pages.

آسام نوتو همان فونت گوگل نوتو می باشد که جهت استفاده در نرم‌افزارهای تحت وب بهینه شده است.

Installation

Using npm 

npm i @aasaam/noto-font

Using CDN 

<link href="https://cdn.jsdelivr.net/npm/@aasaam/noto-font/dist/font-face.min.css" rel="stylesheet">

Then in your CSS file 

body { font-family: "aasaam-Noto"; }
